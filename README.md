# 1Way Home Services

Tax Preparation & Real Estate website for 1Way Home Services — serving El Cajon, San Diego, and surrounding areas.

**Live:** https://1wayhomeservices.com

## Services
- Tax Planning & Advisory
- Tax Filing (Individual & Business)
- Payroll Tax Filing
- Bookkeeping
- IRS Help & Audit Support
- Real Estate Tax Support
- Mortgage Consulting

## Stack
- Next.js 16 (App Router, static export)
- TypeScript
- Tailwind CSS + Framer Motion
- Vercel deployment
